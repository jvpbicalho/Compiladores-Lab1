# Sobre Compiladores

Um compilador é um programa responsável por **traduzir código-fonte escrito em uma linguagem de alto nível** para outra linguagem, geralmente código de máquina.

O processo de compilação normalmente é dividido nas seguintes fases:

1. Análise Léxica  
2. Análise Sintática  
3. Análise Semântica  
4. Geração de Código Intermediário  
5. Otimização de Código  
6. Geração de Código Final
